How It Works

Blockchain Hash Chain

Each status update creates a new hash linked to the previous one, forming an immutable chain.

Role-Based Permissions

Different roles (Manufacturer, Distributor, Retailer) have specific permissions in the supply chain.

Tamper Detection

The system can detect if any record in the chain has been altered or tampered with.
